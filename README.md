    Environment Setup

install PHP >=7.3  
install mongoDB   
Install composer	

    
    Project Setup   
Inside Project Directory (laravel) run below command    
composer install    
php artisan config:clear    
php artisan cache:clear   
php artisan view:clear    
php artisan route:clear   
composer dump-autoload    
php artisan migrate   
    
    Project setup Done!
